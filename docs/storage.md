```typescript
目前`BaseStorage`的数据库结构：key和value 都是buffer。
*key前4个字节为高度，后面为实际的 key[u8a]
[4字节高度 + keyU8a]: valueU8a

然后由于height的高度不为0，故可以用[00, 00, 00, 00]这个来表示最新高度
00 00 00 00 + keyU8a : lastUpdateHeight

例如：
set('key1'/* key */, '1'/* value*/, 1/* height*/);
set('key1'/* key */, '1'/* value*/, 6/* height*/);
set('key2'/* key */, '1'/* value*/, 3/* height*/);
'key1' -> 107, 101, 121, 49
'key2' -> 107, 101, 121, 50
此时数据库结构为：
/** 表示key1的最高高度为 6 */
00, 00, 00, 00, 107, 101, 121, 49    : 00, 00, 01, 10 /** 6 */
/** 表示key2的最高高度为 3 */
00, 00, 00, 00, 107, 101, 121, 50    : 00, 00, 00, 10 /** 3 */
/** 表示key1在高度为1时的值*前四个字节为高度 */
00, 00, 00, 01, 107, 101, 121, 49    : 49
/** 表示key1在高度为6时的值*前四个字节为高度 */
00, 00, 01, 10, 107, 101, 121, 49    : 49
/** 表示key2在高度为3时的值*前四个字节为高度 */
00, 00, 00, 10, 107, 101, 121, 50    : 49


当你开启一个6高度的事务，此时的key2 最新高度是3，然后再组合 3 + keyu8a再去获取值。
```
